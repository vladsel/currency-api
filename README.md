# currency_test_project
Windows program in C++ for obtaining exchange rates using API data of the National Bank of Ukraine (NBU) (https://bank.gov.ua/en/open-data/api-dev).
It must read its configuration from a file or system registry. The configuration should contain parameters such as the interval of receiving exchange rates, the format of saving the received information (JSON, CSV or XML) and any other relevant settings. Received currency information is stored in a file in the format specified in the configuration (JSON, CSV or XML).
